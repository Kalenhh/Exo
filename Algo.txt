Algorithme Jeu du Plus ou Moins

Variable ------------------

valeur : entrée de l'utilisateur
nombre : valeur numérique recherché dans le jeu
n : valeur numérique / compteur à incrémenter dans la boucle du jeu

Lock : Booléen pour bloquer la boucle principal

Algo -----------------------

Tant que Lock est vrai 

demander valeur : "Faire une partie ? y/n"

si valeur est "n"
	Lock devient Faux
	nouveaux tour de boucle

si valeur est "y"
	n <= 0
	nombre <= valeur au hasard entre 1 et 100
	Tant que n est different de 15
		n <= n + 1
		demander valeur : "Proposition de valeur :"
		
		si valeur est égale à nombre 
			afficher : "Gagné en ":n:"tours"
			n <= 15
		
		si valeur est plus petite que nombre 
			afficher : "Trop petit"

		si valeur est plus grand que nombre 
			afficher : "Trop grand"
	
	Fin de la boucle
	
	si valeur est different de nombre 
		afficher : "Nombre de tours depassé , la valeur était ":valeur

sinon 
	afficher : "Entrez y ou n"
